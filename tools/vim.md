# VIM 使用


## 基本操作

1. 按字符移动：`h` `j` `k` `l` 不用说了
2. 按字移动：
    1. `w` -- 向前移动一个字（word）的字首 ；`W` -- 向前移动一个字（忽略特殊字符）的字首
    2. `b` -- 向后移动一个字； `B` -- 向后移动一个字（忽略特殊字符）
    3. `e` -- 向前移动一个字（word）的字尾； `E` -- 向前移动一个字（忽略特殊字符）的字尾
3. 上翻页： `Ctrl+B` （Backward）， 下翻页： `Ctrl+F` （Forward）
4. 行内：`0` -- 行首； `^` -- 行首非空白字符； `$` -- 行尾
